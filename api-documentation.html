Accelerator Data API
Matt Gibbs on May 6th, 2013
The Accelerator Data API is a web service to get data from the LCLS or FACET accelerators. An application (for example, a web page, or iOS app) sends an HTTP request to the Accelerator Data server. The server then parses the request, gets the data from the Channel Access gateway (lcls-prod02), and replies with the requested data in a JSON-formatted object. Currently, there is only one supported action: getting data for an individual PV.

Getting a single PV: GET /PV

To get data for a single process variable (PV), send a GET request to '/PV'.

Parameters:

  • PV: (Required) A String with the name of the process variable you would like to get data for.

Response:

{"units":"mm","PV":"BPMS:LI24:801:X","value":-0.86908829,"timestamp":1367878744521}
  • units: The units for the requested data, i.e. millimeters, GeV, degrees.
  • PV: The PV you requested.
  • value: The current value of the PV.
  • timestamp: The timestamp for the data. This reflects the time at which the PV's data was updated, not the time at which the request was made.
